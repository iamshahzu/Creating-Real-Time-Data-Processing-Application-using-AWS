AWS CodePipeline with Elastic Beanstalk

Learn how to implement continuous delivery using AWS CodePipeline and Elastic Beanstalk

Overview

AWS CodePipeline is a fully managed continuous delivery service that helps you automate your release pipelines for fast and reliable application updates. Elastic Beanstalk is an easy-to-use service for deploying and scaling web applications.
